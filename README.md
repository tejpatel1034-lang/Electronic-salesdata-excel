#  Excel Dashboard Project

A complete Microsoft Excel Dashboard project built for learning **Data Analytics**, **Business Intelligence**, and **Data Science fundamentals** using Excel.

---

#  Project Structure

```
Tej_Patel_ExcelDashboard.xlsx
│
├── Raw_Data
├── Clean_Data
├── Analysis
├── Summary output
├── Pivot_Reports
├── Lists
├── Dashboard
└── Documentation
```

---

#  Worksheets Description

##  Raw_Data
- Original dataset
- No modifications
- Source data for the project

---

## Clean_Data
Data cleaning performed using Excel.

Tasks completed:
- Removed duplicate records
- Handled blank values
- Fixed formatting
- Standardized data
- Prepared data for analysis

---

##  Analysis
Business calculations and KPIs.

Includes:
- Total Sales
- Profit Analysis
- Average Sales
- Category Analysis
- Region Analysis
- Customer Analysis

---

##  Summary output
Quick business summary including:

- Total Revenue
- Total Profit
- Number of Orders
- Average Order Value
- Best Performing Category
- Best Region

---

##  Pivot_Reports
Interactive Pivot Tables for business insights.

Examples:
- Sales by Region
- Sales by Category
- Monthly Sales
- Customer Analysis
- Profit Analysis

---

##  Lists
Contains helper lists used for:

- Data Validation
- Drop-down menus
- Dynamic selections
